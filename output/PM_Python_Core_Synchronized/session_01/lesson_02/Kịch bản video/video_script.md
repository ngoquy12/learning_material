- **Thời lượng dự kiến**: 5-7 phút
- **Loại bài giảng**: Bài học thực hành (Practice lesson)
- **Nhạc nền đề xuất**: Lofi Chill không lời (âm lượng -20dB, tốc độ 1.5x)
- **Chuẩn hình ảnh**: Full HD 1080p, tỷ lệ 16:9, font chữ đồng bộ thương hiệu Rikkei Education

## 1. Mục tiêu học tập (Learning Objectives)
1. Giới thiệu bài toán thực hành cho 'Cài đặt môi trường'.
2. Mô hình hóa các bước thực hành qua terminal / IDE, hướng dẫn viết code trực tiếp, khởi chạy uvicorn, và kiểm tra đầu ra.
3. Tổng kết checklist hoàn thành bài Lab thực hành.
