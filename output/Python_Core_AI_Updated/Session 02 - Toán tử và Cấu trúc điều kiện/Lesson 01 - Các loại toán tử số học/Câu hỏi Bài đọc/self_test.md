# Bộ câu hỏi Khảo thí & Đánh giá năng lực: Các loại toán tử số học

### Câu 1: Phép chia lấy nguyên (//) sẽ trả về kết quả như thế nào với số âm, ví dụ -5 // 2?

**Gợi ý trả lời & Hướng dẫn tự học:**
Phép chia lấy nguyên luôn làm tròn xuống số nguyên gần nhất nhỏ hơn hoặc bằng kết quả thực tế. -5 / 2 bằng -2.5, làm tròn xuống (về phía âm vô cực) sẽ là -3.
