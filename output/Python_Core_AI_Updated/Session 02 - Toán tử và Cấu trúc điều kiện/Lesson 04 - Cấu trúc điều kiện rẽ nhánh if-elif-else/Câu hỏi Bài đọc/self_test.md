# Bộ câu hỏi Khảo thí & Đánh giá năng lực: Cấu trúc điều kiện rẽ nhánh if-elif-else

### Câu 1: Làm thế nào để tránh lỗi Unreachable Code khi sử dụng cấu trúc if-elif-else?

**Gợi ý trả lời & Hướng dẫn tự học:**
Sắp xếp thứ tự kiểm tra từ điều kiện nghiêm ngặt nhất (phạm vi hẹp nhất) xuống thấp dần. Tránh để điều kiện bao quát nằm phía trên điều kiện cụ thể.

---

### Câu 2: Nhánh else trong cấu trúc if-elif-else có bắt buộc phải xuất hiện không?

**Gợi ý trả lời & Hướng dẫn tự học:**
Không bắt buộc. Nhánh else chỉ cần thiết khi bạn muốn xử lý tất cả các trường hợp còn lại nếu tất cả các điều kiện if và elif phía trước đều nhận kết quả False.
