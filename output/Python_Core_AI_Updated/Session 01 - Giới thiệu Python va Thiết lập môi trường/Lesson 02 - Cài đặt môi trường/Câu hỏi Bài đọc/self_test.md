# Bộ câu hỏi Khảo thí & Đánh giá năng lực: Cài đặt môi trường

### Câu 1: Tại sao phải thêm Python vào biến môi trường PATH khi cài đặt?

**Gợi ý trả lời & Hướng dẫn tự học:**
Biến hệ thống PATH cho phép hệ điều hành tìm thấy tệp thực thi của Python từ bất kỳ thư mục nào thông qua giao diện dòng lệnh Terminal hoặc Command Prompt mà không cần chỉ định đường dẫn tuyệt đối đầy đủ đến thư mục cài đặt gốc.

---

### Câu 2: Môi trường ảo venv hoạt động như thế nào để cô lập các dự án Python?

**Gợi ý trả lời & Hướng dẫn tự học:**
venv tạo ra một bản sao thu nhỏ của thư mục cài đặt Python gốc trong thư mục dự án của bạn. Khi được kích hoạt, nó sẽ sửa đổi biến môi trường PATH tạm thời trong phiên làm việc của dòng lệnh để ưu tiên tìm kiếm và sử dụng Python Interpreter cùng các thư viện cài đặt trong thư mục ảo này.
