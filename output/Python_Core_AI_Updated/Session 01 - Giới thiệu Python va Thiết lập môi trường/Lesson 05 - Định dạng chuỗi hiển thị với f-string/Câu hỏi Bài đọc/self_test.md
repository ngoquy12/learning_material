# Bộ câu hỏi Khảo thí & Đánh giá năng lực: Định dạng chuỗi hiển thị với f-string

### Câu 1: Làm thế nào để in ra đúng ký tự ngoặc nhọn { hoặc } khi đang sử dụng f-string?

**Gợi ý trả lời & Hướng dẫn tự học:**
Để in ký tự ngoặc nhọn vật lý trong f-string mà không kích hoạt trình xử lý biểu thức, bạn chỉ cần viết đúp hai ký tự ngoặc nhọn liên tiếp: dùng {{ để hiển thị { và dùng }} để hiển thị }.

---

### Câu 2: Sự khác biệt giữa căn lề có ký tự và không có ký tự định dạng là gì?

**Gợi ý trả lời & Hướng dẫn tự học:**
Khi sử dụng {name:10}, Python mặc định căn lề trái đối với chuỗi. Tuy nhiên, khai báo rõ ràng bằng cách dùng các ký hiệu như {name:<10} giúp mã nguồn dễ hiểu và đảm bảo tính nhất quán giữa các kiểu dữ liệu khác nhau.
