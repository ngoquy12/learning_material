Giới thiệu ngôn ngữ Python
Biên dịch & Thông dịch

Biên dịch (Compiled Language)

01 Quét toàn bộ mã nguồn (.cpp, .java)
02 Dịch ra file thực thi nhị phân (.exe, .bin)
03 Thực thi trực tiếp bởi hệ điều hành

Thông dịch (Interpreted Language)

01 Trình thông dịch đọc code Python (.py)
02 Chuyển sang dạng bytecode trung gian
03 Máy ảo Python (PVM) thực thi từng dòng
