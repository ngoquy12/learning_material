🛡️ Prerequisite Guard Agent

Xây dựng DAG dependency graph, phát hiện vi phạm tiên quyết trước khi tốn bất kỳ token nào. BLOCKER → dừng pipeline.

🧠 Knowledge Memory Agent

SQLite-backed memory store phân loại 10 loại lỗi. Creator Agent query trước khi sinh nội dung — không bao giờ lặp lại lỗi cũ.

💾 Semantic Cache Layer

TF-IDF fuzzy matching (88% threshold) + SHA-256 exact match. Tiết kiệm ~40% token cost. Tắt/bật qua SEMANTIC_CACHE_ENABLED.

📦 SCORM 1.2 Export

Xuất toàn bộ học liệu thành file .zip chuẩn SCORM 1.2, nạp trực tiếp vào Moodle, Canvas, Blackboard, TalentLMS.

🔗 Obsidian Knowledge Vault

Graph View với prerequisite frontmatter, Prerequisite Map và bidirectional WikiLinks — bản đồ lộ trình học tập trực quan.

🔄 Self-Correction Loop

Mỗi artifact có vòng lặp Reviewer tự động sửa lỗi (tối đa 3 lần). Lỗi được lưu vào Knowledge Memory để phòng tránh tương lai.
